## 주기능
* 소셜 로그인
* 그룹 생성, 편집, 삭제
* 그룹 이름 및 태그 검색, 맛집 검색
* 그룹 팔로우, 언팔로우
* 그룹에 코멘트와 함께 맛집 저장
* 반응형 웹페이지
